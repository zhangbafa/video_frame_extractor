# Video Frame Extractor - 视频抽帧工具使用说明

## 项目简介

Video Frame Extractor 是一个高效的命令行视频抽帧工具，支持多种抽帧模式和批量处理功能。适用于视频编辑师、内容创作者、数据分析师和研究人员等需要从视频中提取关键帧的用户。

## 功能特点

- 支持多种抽帧模式：
  - 时间间隔抽帧（每隔指定秒数抽取一帧）
  - 固定帧数抽帧（均匀抽取指定数量的帧）
  - 智能关键帧检测（自动识别场景变化明显的帧）
- 支持多种输出格式（JPG、PNG）和质量设置
- 批量处理多个视频文件
- 简洁直观的命令行界面
- 交互式模式支持

## 快速开始

### 方法一：直接运行（无需安装）

1. 确保已安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行主程序：
   ```bash
   python main.py --help
   ```

### 方法二：安装为命令行工具

1. 安装包：
   ```bash
   pip install .
   ```

2. 使用命令行工具：
   ```bash
   video-extract --help
   ```

## 基本用法示例

### 1. 时间间隔抽帧

每5秒提取一帧，保存为jpg格式：
```bash
# 安装后使用
video-extract -i video.mp4 -o frames -m time -t 5 -f jpg

# 直接运行
python main.py -i video.mp4 -o frames -m time -t 5 -f jpg
```

### 2. 固定帧数抽帧

提取10帧，均匀分布在视频中：
```bash
video-extract -i video.mp4 -o frames -m count -n 10
```

### 3. 关键帧检测

自动检测关键帧，灵敏度为70（值越高检测越敏感）：
```bash
video-extract -i video.mp4 -o frames -m keyframe -s 70
```

### 4. 批量处理

处理目录下所有视频：
```bash
video-extract -i videos/ -o frames/ -m time -t 10 -r
```

处理多个指定视频：
```bash
video-extract -i video1.mp4 video2.mp4 -o frames/ -m count -n 20
```

## 交互式模式

如果不想使用命令行参数，可以直接运行程序进入交互式模式：

```bash
# 安装后使用
video-extract

# 直接运行
python main.py
```

程序会引导您逐步输入所需信息，适合初次使用的用户。

## 测试功能

项目包含一个测试脚本，可以快速测试三种抽帧模式的效果：

```bash
python test_extractor.py
```

运行后，按照提示输入视频路径，程序会自动测试三种抽帧模式并将结果保存到 `test_output` 目录。

## 创建测试视频

如果您没有测试视频，可以使用项目提供的脚本创建一个简单的测试视频：

```bash
python create_test_video.py
```

这将在 `test_video` 目录下创建一个名为 `sample.mp4` 的测试视频。

## 命令行参数详解

```
usage: video-extract [-h] [-i INPUT [INPUT ...]] [-o OUTPUT] [-m {time,count,keyframe}]
                     [-f {jpg,png}] [-q QUALITY] [-t INTERVAL] [-n NUMBER]
                     [-s SENSITIVITY] [-r] [-v] [-y]

视频抽帧工具

options:
  -h, --help            show this help message and exit
  -i INPUT [INPUT ...], --input INPUT [INPUT ...]
                        输入视频文件路径或目录
  -o OUTPUT, --output OUTPUT
                        输出目录路径
  -m {time,count,keyframe}, --mode {time,count,keyframe}
                        抽帧模式
  -f {jpg,png}, --format {jpg,png}
                        输出图像格式
  -q QUALITY, --quality QUALITY
                        输出图像质量(1-100，仅jpg有效)
  -t INTERVAL, --interval INTERVAL
                        时间间隔(秒)，用于time模式
  -n NUMBER, --number NUMBER
                        提取帧数，用于count模式
  -s SENSITIVITY, --sensitivity SENSITIVITY
                        关键帧检测灵敏度(1-100)，用于keyframe模式
  -r, --recursive       递归处理目录下所有视频文件
  -v, --verbose         显示详细信息
  -y, --yes             自动确认覆盖已存在文件
```

## 高级用法

### 1. 高质量PNG输出

```bash
video-extract -i video.mp4 -o frames -m time -t 1 -f png
```

### 2. 高压缩比JPG输出

```bash
video-extract -i video.mp4 -o frames -m count -n 50 -f jpg -q 70
```

### 3. 详细输出模式

```bash
video-extract -i video.mp4 -o frames -m keyframe -s 60 -v
```

### 4. 递归处理嵌套目录

```bash
video-extract -i videos_collection/ -o all_frames/ -m time -t 30 -r
```

## 常见问题

### 1. 程序运行时报错 "No module named 'cv2'"
这表示您尚未安装OpenCV。请运行以下命令安装所需依赖：
```bash
pip install -r requirements.txt
```

### 2. 关键帧检测不够准确
您可以调整关键帧检测的灵敏度参数 `-s`，值越高检测越敏感：
```bash
video-extract -i video.mp4 -o frames -m keyframe -s 80
```

### 3. 处理大型视频文件时内存占用过高
对于大型视频文件，建议使用时间间隔模式并设置较大的间隔值，或使用固定帧数模式限制提取的帧数。
